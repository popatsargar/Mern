Quiz on Basic HTML Tags

  1. Which tag is used to create a hyperlink in HTML?

    1. <img>
    2. <div>
    3. <a>
    4. <link>

  2. Which tag is used to display an image?

    1. <picture>
    2. <img>
    3. <src>
    4. <figure>

  3. Which tag is used to create a numbered list?

    1. <ul>
    2. <dl>
    3. <menu>
    4. <ol>

  4. Which HTML tag is used for the largest heading?

    1. <h3>
    2. <h4>
    3. <h5>
    4. <h1>

  5. Which tag is used to insert a line break?

    1. <p>
    2. <br>
    3. <hr>
    4. <break>

  6. Which tag is used to create a table row?

    1. <td>
    2. <th>
    3. <tr>
    4. <table>

  7. Which HTML tag is used to display bold text?

    1. <em>
    2. <i>
    3. <b>
    4. <strong>

  8. Which tag is used to define an unordered list?

    1. <ul>
    2. <ol>
    3. <list>
    4. <dl>

  9. Which tag is used for inserting a horizontal line?

    1. <line>
    2. <br>
    3. <div>
    4. <hr>

  10. Which tag is used to define the main content section of an HTML page?

    1. <section>
    2. <div>
    3. <main>
    4. <body>
